    What Are Computers ?

    It is A Electronic Machine which take input from user and make Output From That Input. If set of Instruction is given then it is called Program

    Computer Basic Components

    • CPU: It is Brain Of Computer Which Processes The Instruction.
    • RAM: It is Temporary Memory Of Computer. It is fast , But Expensive.
    • Monitor: It is a device which Shows What is Current Events

    Properties Of Computer

    • Speed: It can Solve Large & Complex Problems in Few Seconds
    • Storage: It is Capisity To Store Data in Computer. It is generally divided into two categories, main memory and secondary storage.
    • Accuracy: Computer calculations have great accuracy. The accuracy achieved by its hardware configuration and the instructions
    • Reliability : A computer produces results with no error. Most of the errors generated are human errors that are created by the user itself.
    • Versatility : Computers are versatile machines. They can perform many different tasks.
