      • Go to Python Official Website Click Here To Visit

      • Then There Will Be Option For Windows Click There Click Here To Skip All Steps
      • Then From Stable Version Download The Latest Version
      • Then A .exe File Downloaded To Your System , Run That .exe File
      • When That .exe File is Opened Tik The Two Boxes Below
      • Then Install The Python In Your System
      • Now Python is Installed
      • To Check That Python is Installed. Below are Some Steps :
        • Press Windows + R
        • Type 'Cmd'
        • Then Command Box Will Open
        • Type 'python --version' To Check
          • If Python Is Installed Then It will Show Version In CMD Box
          • If Python is Not Installed Then It will Not Work, Retry Install

    Powershell

    • Press Windows + R
    • Type 'powershell' and Click Ok
    • Then Type 'winget install -e --id Python.Python.3.xx --scope machine'
    • Replace xx With The Latest Version Of Python You Know
    • To Check That Python is Installed. Below are Some Steps :
